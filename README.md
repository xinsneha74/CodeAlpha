# Real Estate Market Trends Dashboard

## Project Overview

This project analyzes real estate market trends using Power BI. The dashboard helps investors, developers, and analysts understand property prices, rental yields, market demand, and investment opportunities.

## Objective

To build an interactive dashboard that provides insights into real estate market dynamics and supports data-driven investment decisions.

## Dataset Information

The dataset contains information about:

* Property ID
* City
* Property Type
* Property Price
* Monthly Rent
* Latitude
* Longitude

## Key Performance Indicators (KPIs)

* Average Property Price
* Average Monthly Rent
* Rental Yield (%)
* Total Properties Listed
* City-wise Property Analysis

## Dashboard Features

### 1. Market Overview

* Average Property Price
* Average Rent
* Total Listings
* Property Type Distribution

### 2. Price Analysis

* Property Price by City
* Property Price Trend
* Top Expensive Locations

### 3. Rental Yield Analysis

Rental Yield (%) = (Annual Rent / Property Price) × 100

### 4. Geographic Analysis

* Location-based Property Mapping
* Heat Maps
* Investment Hotspots

### 5. Investment Insights

* High Rental Yield Areas
* Best Cities for Investment
* Market Growth Opportunities

## Tools Used

* Power BI
* Microsoft Excel
* DAX

## Key Insights

* Identified cities with the highest property prices.
* Compared rental income across locations.
* Visualized investment hotspots using geographic analysis.
* Evaluated rental yield performance for investment planning.

## Outcome

The dashboard enables users to monitor market trends, identify profitable locations, and make informed real estate investment decisions.
